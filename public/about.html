Tools Used

Foundation

This is a CSS framework we used to make the website more appealing than plain HTML would. It also helped use make the filter layout responsive.

Vue

This is a popular JS framework used to make reactive web applications. We used it for binding all interactive page elements to ref variables.

Leaflet API

This is a JS library for creating interactive maps. We used it for the crime map at the top of the page.

Nominatim API

This API is used for converting addresses to coordinates and back again. We used it to make the address bar above the map accurately reflect position, and to correctly move to new locations.

Express

This is a Node.js web server framework. We used it primarily to implement the REST API needed for the crime statistics.

SQLite3

This is a simple SQL database system. We used it to store the details on all crimes, neighborhoods, and codes.

cors

This is a simple Node.js library used in conjunction with Express. We used it so that we could make requests from the website at localhost:5173, to the REST server at localhost:8000
